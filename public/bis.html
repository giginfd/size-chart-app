Omnisend Sync Helper

Use this only when Omnisend back-in-stock counts need to be pulled again from Omnisend.

Manual step needed
Important: This is different from clicking Refresh BIS Data. Omnisend must be synced first if you want fresh Omnisend counts.
Install Sync Bookmark Open Omnisend Report
1

Install the bookmark once

Click Install Sync Bookmark, then save it to your bookmarks bar. You only need to do this the first time.

2

Open the Omnisend Products Report

Open the Omnisend report page and wait until it fully loads before doing anything else.

3

Run your saved SYNC BIS BOOKMARK

Click the bookmark while you are on the Omnisend page. It can take around 1 to 2 minutes. Do not close the page. Wait for the confirmation popup.

4

Come back here and refresh

After Omnisend finishes syncing, return to this page and click Refresh BIS Data to load the newest cache into the dashboard.

Quick rule: Omnisend sync first, then Refresh BIS Data here.
