How to refresh Omnisend BIS data
First time only:
Install Sync BIS Bookmark
Drag this link to your bookmarks bar, or open it and save it as a bookmark.

To refresh data:
1. Open Omnisend BIS Report and wait for the page to fully load.
2. Click your saved SYNC BIS BOOKMARK while on the Omnisend page. It might take 2 minutes to sync all data, don't close the page and wait for a confirmation pop-up.
3. Return here — the dashboard will show the updated data.
Idle - OK
If a refresh is running, wait for it to finish. The table updates automatically when the new cache is ready.
